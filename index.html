What it does

Urban Nest Hub brings all your selling channels into one place — reducing manual work, preventing errors, and letting you scale without friction.

01

Bulk Listing Creation

Create and publish product listings across multiple platforms simultaneously. Import from CSV, Word exports, or Vinted files in seconds.

02

Order Management

View and process orders from all connected channels in a unified inbox. Mark as dispatched, refunded, or returned with one click.

03

Inventory Sync

Stock levels and pricing are kept in sync across platforms automatically, preventing overselling and manual update errors.

04

Image Management

Reverse image search integration with Google Lens. Bulk image download, upload, and optimisation built directly into the workflow.

05

Shipping Labels

Generate and print shipping labels and QR codes without leaving the application. Integrated with eBay's fulfilment system.

06

TikTok Shop Integration

Push listings directly to TikTok Shop, manage your catalogue, and sync orders — all from within the Urban Nest Hub interface.

Privacy Policy

Last updated: 25 May 2026

1. Who We Are

Urban Nest Hub is a desktop software application developed by Efficient Repairs and Installations (Company No. 13444965), trading as Urban Nest Hub, registered at Suite 11 (17B), Bridgewater House, Surrey Road, Nelson, Lancashire, BB9 7TZ, United Kingdom. Contact: faz@efficientboilerrepairs.co.uk.

2. What Data We Access

Urban Nest Hub connects to third-party selling platforms (including eBay, Vinted, and TikTok Shop) on behalf of the seller who installs and uses the application. The application may access the following categories of data through authorised platform APIs:

  • Product listing data (titles, descriptions, images, prices, categories)
  • Order information (buyer name, delivery address, order status)
  • Inventory and stock levels
  • Seller account information required for authentication
  • Messaging data (where available via platform API)

This data is accessed solely to provide the features of the Urban Nest Hub application to the licensed user (the seller). We do not sell, share, or transfer this data to any third party.

3. How Data is Stored

All data accessed by Urban Nest Hub is stored locally on the user's own computer. No seller data, product data, or order data is transmitted to or stored on Urban Nest Hub servers. API credentials (access tokens) are stored locally in encrypted configuration files on the user's machine.

4. TikTok Shop Data

When a user authorises Urban Nest Hub to connect to their TikTok Shop account, the application will access product catalogue data, listing information, and order data via the TikTok Shop Open API. This access is limited to the permissions explicitly granted by the user during the OAuth authorisation flow. Urban Nest Hub uses this access solely to:

  • Create and update product listings on TikTok Shop on behalf of the seller
  • Retrieve order information for display within the application
  • Synchronise inventory and pricing data

Urban Nest Hub does not use TikTok Shop data for advertising, profiling, or any purpose beyond the seller's own listing management.

5. eBay Data

Urban Nest Hub connects to the eBay Sell and Buy APIs using OAuth 2.0 tokens authorised by the seller. Data accessed includes listings, orders, messages, and account details. This data is used exclusively within the application for the benefit of the authorised seller and is not shared with third parties.

6. Licence Validation

Urban Nest Hub performs licence key validation by making a read-only request to a publicly hosted licence file. This request transmits no personal data and is used solely to verify that the installed copy of the software is properly licensed.

7. Third-Party Services

Urban Nest Hub may integrate with the following third-party services as directed by the user:

  • eBay — eBay Developer Program API
  • TikTok Shop — TikTok Shop Open Platform API
  • Vinted — Data import from exported files
  • Google Lens — Image search (browser-based, no data stored)
  • Stripe — Payment processing for software subscriptions (Stripe's own privacy policy applies)
