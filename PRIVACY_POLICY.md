**Last updated: 5 May 2026**

## 2. What Data We Collect

Urban Nest Hub connects to selling platform APIs (including TikTok Shop and eBay) to retrieve and manage the following data on behalf of the seller:

- **Order data** — order IDs, item details, order status, fulfilment information
- **Buyer usernames** — platform usernames associated with orders (e.g. eBay buyer ID, TikTok Shop buyer ID)
- **Listing data** — product titles, prices, descriptions, SKUs, stock levels
- **Messages** — buyer-seller messages retrieved from platform inboxes
- **API credentials** — access tokens and refresh tokens used to authenticate with platform APIs

We do **not** collect:

- Buyer payment information or financial data
- Buyer real names, home addresses or contact details beyond what platforms provide in order data
- Any data from users who are not the operator of the application

---

## 3. How We Use Data

Data retrieved by Urban Nest Hub is used solely for the following operational purposes:

- Displaying and managing active listings on connected platforms
- Viewing and processing orders
- Responding to buyer messages
- Managing inventory levels
- Generating internal reports for the seller's own use

We do **not** use data for advertising, profiling, or any purpose beyond the direct management of the seller's own selling accounts.

---

## 4. How Data Is Stored

Urban Nest Hub is a **locally installed desktop application**. All data is stored exclusively on the operator's local device:

- No buyer or order data is transmitted to or stored on any external server owned or operated by Urban Nest.
- API credentials and tokens are stored in local configuration files on the operator's device, protected by operating system access controls.
- All communications with platform APIs are encrypted using HTTPS/TLS.
- The application does not operate any web server or cloud storage service.

---

## 5. Data Sharing

We do **not** sell, rent, or share any data with third parties. Data retrieved from platform APIs is used solely within the Urban Nest Hub application by the authorised operator.

The application communicates directly and exclusively with the following platforms via their official APIs:

- TikTok Shop Open API
- eBay Sell API

---

## 6. Data Retention

- **Active use:** Data is retained locally on the operator's device for as long as the application is in active use.
- **End of use:** Upon discontinuing use of the application, all locally stored data can be deleted by removing the application and its associated configuration files from the device.
- **Platform request:** Upon request from TikTok Shop or any connected platform, all data collected under that platform relationship will be permanently deleted from local storage within **30 days**.
- **End of contract:** At the end of any contractual relationship with TikTok Shop, all associated seller and buyer data will be permanently deleted.

---

## 7. Your Rights (UK GDPR)

Under the UK General Data Protection Regulation (UK GDPR) and the Data Protection Act 2018, individuals have the following rights regarding their personal data:

- **Right of access** — to request a copy of any personal data held
- **Right to rectification** — to request correction of inaccurate data
- **Right to erasure** — to request deletion of personal data
- **Right to restriction** — to request that processing be restricted
- **Right to object** — to object to processing of personal data
- **Right to data portability** — to receive data in a machine-readable format

To exercise any of these rights, please contact us at the email address above. We will respond within **30 days**.

---

## 8. Data Breach Notification

In the event of a confirmed or suspected data breach involving personal data:

- We will assess the breach within **24 hours** of discovery.
- Affected platforms (including TikTok Shop) and any affected sellers will be notified within **72 hours** of confirmation.
- The UK Information Commissioner's Office (ICO) will be notified where required under UK GDPR.
- API credentials will be revoked immediately upon discovery of a suspected breach.
